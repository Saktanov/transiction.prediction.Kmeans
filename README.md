# Transaction Clustering

## Описание проекта

Данный проект представляет собой систему кластеризации банковских транзакций с использованием алгоритма машинного обучения.

Модель анализирует транзакции на основе таких признаков, как сумма и географические координаты (широта и долгота), и распределяет их по кластерам. Проект включает обучение модели, предсказание через консоль и интерактивное веб-приложение на Streamlit.

## Цель проекта

Цель проекта — выявление закономерностей в транзакциях, сегментация данных и упрощение анализа финансовой активности с помощью кластеризации.

## Установка и запуск

### 1. Клонировать репозиторий

git clone https://github.com/your-username/transaction-clustering.git
cd transaction-clustering

### 2. Создать виртуальное окружение

python -m venv .venv
source .venv/bin/activate   # для Linux/Mac
.venv\Scripts\activate      # для Windows

### 3. Установить зависимости

pip install -r requirements.txt

### 4. Обучить модель

python train_model.py

После выполнения будут использованы/обновлены:

* kmeans_model.pkl
* scaler.pkl

### 5. Запуск CLI-предсказания

python predict_console.py

### 6. Запуск веб-приложения

streamlit run app.py

### 7. Открыть в браузере

http://localhost:8501

## Использование

Проект поддерживает следующие сценарии:

* обучение модели кластеризации

## Как работает модель

В проекте используется:

* StandardScaler — нормализация данных
* KMeans — кластеризация транзакций
* проверка входных данных
* визуализация кластеров
